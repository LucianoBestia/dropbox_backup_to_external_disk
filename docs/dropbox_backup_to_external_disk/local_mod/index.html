move_or_rename_local_files

Files are often moved or renamed After compare, the same file (with different path or name) will be in the list_for_trash and in the list_for_download. First for every trash line, we search list_for_download for same size and modified. If found, get the remote_metadata with content_hash and calculate local_content_hash. If they are equal move or rename, else leave: it will be trashed and downloaded eventually. Remove also the lines in files list_for_trash and list_for_download.
